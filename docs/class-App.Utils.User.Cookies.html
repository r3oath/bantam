Create/set a new cookie in the users browser. This cookie will be automatically encrypted to provide futher protection against someone stealing the users cookie(s) and attempting to read out the content.

The name of the cookie.

The value for the cookie (will be encrypted).

When the cookie will expire, default is in 5 years time from the time it was set.

True if the cookie was set, false otherwise (does not indicate whether the user accepted the cookie though).
