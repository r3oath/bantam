Class Route

Provides basic URL routing and URL token extraction.

// Register a new route for /users/ grabbing the username provided and print it out.
Route::register('users/:name', function(){
    $name = Route::getToken('name');
    echo $name;
});

# register( string $signature, Closure $closure, $regexs = [] )

Registers a new route with the routing table and specifies a callback for that route. You can specify capture tokens by prepending the : character to the keyword, for example, if you create a users route 'users/:name', and the url 'yoursite.com/users/bob' is accessed, the token 'bob' will be captured against ':name' and will be available to you through the Route::getToken('name') helper.
